窗沿里暗色的水珠窸静地汇聚，凌晨时分的深蓝城市扩散出霓色的潋滟靡丽。

「呐，咱们现在……是要做什么？」

窗影上出现一个举着双手的少年，由于身前的阻碍，头颈向后微微仰去，柔韧成一道白鹤般的优雅弧线。

只不过有时甚不老实地左摆右晃歪头就是了。

他面前正与围巾的一百零八种系法打交道以及与衣服的对称度持续战斗的女生头也不抬。

边说着她又将刚系好的围巾又拆了，端着下巴纳罕地打量起他。

「不对，这样不行。」

望着又要陷入纠结与深思的自家审神者，白发付丧神会意地放下胳膊，舒展地斜倒进一旁椅子里，趴在椅背上用透彻又清冽的金眸子瞧她。

「所以啊，时政发布的任务呢，为什么你这次什么都不说？」

「还是说……」

他淌了蜜的深秋色眼睛似笑非笑注视着在透蓝色屏幕上搜寻资料的她。

「你这次……是有什么难言之隐吗？」

女生飞快移动着的眼球有一瞬凝滞。

「果然……鹤先生还是长的太好看了，」她盯视住一张图片使其自动增长分辨率后，语调突然升高，透出平素语气里未曾掺过的嫌弃，「我明明已经按这个时代的最低要求来搭配了，结果……」

她的脸忽然在面前放大，手指拎起他的额边碎发后却沿着发线逐渐下移，刀削的冰刻下巴被她温热的手指轻轻点触。

她专注认真又毫无机质的深黑瞳孔里，曵起了屋外淅沥的雨声。

「呐，近侍大人……我可以擅自改变你的模样吗。」

容貌昳丽的白色少年沉静地盯了她几许，随即就微眯眼睛疏狂地笑起来。

「嘛……既然你这么有自信的话，我就尽情享受惊喜了。」

她淡淡的目光里亦映出了温意。

「承蒙信任，不胜感激。」

「你刚刚才说过信任我的。」两分委屈。

溪流里的金沙，亦仿佛流的更欢了。

一抹珍珠白旋转着出现在色彩纷繁的众伞中央。

举着伞柄的手是健康的小麦色，随着行走与风拂而微微扬起的是鸦黑的短发。时刻蕴着温笑的眼睛里反射出的是墨色琉璃的光彩。

「这可真是吓到我了，你确定你真的不会遭到灵力反噬吗？」

少年如水滴一样透亮的声音清凉凉钻进身旁同样乌眉黛眼人的耳里。

她眉眼自信地扬起。

「没问题的，这是假象。我只是利用灵力暂且改变了你身上一些东西的频率，不过是糊弄人眼的东西而已。」

「不过……鹤先生真是难办啊，就算变成这样低调的样子，却还是能吸引住很多纯情少女的目光呢。 」

她带着笑意和揶揄看向他身后不少闪着惊艳的亮晶晶眼神。

「喂喂，饶了我吧，你把我的头发布置的连脸都快遮没了，而且我穿的不也是你所谓的大叔装吗～」

他向外竖了竖土色工装外套的立领，却不小心扯松了扣子，深色的内衬与美学顶级形状的锁骨被扯现出来。虽已变成低调不再雪白流光的小麦黄肤色，却反而让他生出一种平素不曾显露的野性吸引力。

她默了默，无奈地还是伸出手决定替他系回去。

他微屈身配合她，一侧没握着伞的手将她故意拨到他额前的几丝碎发轻撩开来，向她轻快地眨眨眼：「有什么关系，反正时政会把他们的记忆都替换掉不是吗？」

不同以往的墨色眼睛于近在咫尺的地方清澈透亮，浅淡鎏金色火焰自那丝绒一般的深色幕布下微微风动。

奇怪。

怎么她的引擎就没搜索到，这人瞳色变深之后，原本淡色瞳孔能掩盖的情绪，现在似是遮不住了。

轻快、俏皮、戏谑、却又专注、镇定、温柔，仿佛全世界就只能看见她一个人的、一种奇特异样的隐藏情绪。

正被他眼眸里浓的落墨的色调。

——点点滴滴地出卖着。

时空有瞬间的寂静，谁耳边叮咚的雨声仿佛被莫名生长的屏障隔离殆尽。

一句不知怎得生出、想要强烈表达出的『有关系』在喉咙里来回窜行了数次。

最终还是消弭于胸腔深处。

她只是一本正经地开口。

「……总是人为干预大脑的记忆，他们会变傻的。」

「哦？原来是这样，那么我就照顾一下他们好了。」

眼见面前少年模样的神祗开始十分自觉地回揽自己的衣襟与领扣，她低下头藏起自己低低翘起的嘴角。

她点触手上腕表查看时间，恰好又错过头上温热悠长投来的沉金色。

屏幕灵敏飞快地切换，最终定格于指向圆周四分之三处的莹蓝指针。

再抬起头时她已是略有歉色的笑意，他也迅速转回四处张望的视线温温看她。

「不好意思鹤先生，上课时间要到了。」

说完她握住旁边那人故意歪向她大半的伞柄下侧，边向反方向微微推动，边向前轻拽起尾端。

她加快步伐回头笑起来。

「鹤大叔，再不快点，我就要把伞抢走了哦。」

\ \ \ \ \ \ \ \ \ \ \

宽敞明亮的阶梯教室里虽已喧闹一片，陆续走进的人身上却也多少沾着秋霖的冰凉，浓度适宜的烟火气一寸寸温然蔓延开来。

鹤丸国永亦将手里皎白的伞学着众人的样子撑开落于教室外的空阔处。

调整好她安置在自己耳里的隐形助译器，确认脑海里进入产出的都是上世纪的标准语。思维中回荡着那人方才递过来的寥寥信息，他扯出一抹无奈的笑意。

『嘛，无论如何，让鹤来和20岁的异国年轻人称兄道弟……也太过于相信我这个平安老头子了吧。』

然而他转过身来就径直向前面一排的五个人爽快地凑了过去，拍拍其中一个人的肩膀。

「哟！不好意思我来晚了，怎么样，老师还没来吧？」

那人笑呵呵搂住他肩膀，他还更加亲热地扶了回去。

「诶呀是鹤丸啊～」

另一边的少年温和地向他笑笑。

「坐吧鹤丸，老师还没来。」

还有一个正奋笔疾书的，见他来眼睛一亮。

「诶鹤丸老兄，上周的作业做的怎么样？」

他微愣几秒，随即淡定回复。

「哦那个啊！……做完了，还算简单，怎么了？」

「是嘛是嘛！来和我对一下吧——」

「没问题～」

对21世纪年轻大学生领域一无所知的鹤丸国永同学，捧着自家审神者事先准备好的一大摞资料。对着五个名义上的舍友，笑容可掬地胸有成竹。

\ \ \ \ \ \ \ \ \ \ \

距离上课还有半分钟。

和『舍友们』相谈甚欢的鹤丸忽然似有所感地转头。

精准地踏着清脆急促的铃音，一个身影凌冽地翻了进来。

坐在靠边第二个位置的鹤丸下意识敞开了臂膀。

满怀的霜冷风雨与温热柔软。

他一旁的少年眼尖地发现了他怀里多出来的人，笑眯眯地将大半部身子挪到左边自动消音也将好奇的其他人隐了视线。

「好了好了老师还没来，咱一起上会自习别看了别看啦～」

鹤丸感激地看了对方一眼。

回眸发现怀中人浑身渗发着的冰雪气似乎越来越浓，甚至有蔓延加剧的态势。而体重似乎也愈来愈轻，几近于无。

他方要低声询问她如何了，却见她紧闭的黑眸遽然而睁。

异样皆数消失。

她尚还脱力趴于他胸前，却忽清清淡淡笑起来。

「怎么了？抱我抱上瘾了吗。」

鹤丸国永默而不语盯了她许久，直至门外高跟鞋的啪嗒声叠叠响起。

他才敛起再次不小心流露碎金的黑瞳，轻轻将她扶到桌上稳定借力。

「下次……不必这么拼。」

她将气力恢复些的身姿微微调整坐正时，耳边低低柔柔钻进了有些气馁的音节，额边发丝被微凉的指尖徐徐地挽起。

她看向近在咫尺、表情里难得透出肃穆又挫败的颜色的他。

是基于系统软件底层才能顺利运行的一个普通的应用软件而已。

只因内核是由工程师亲自创新开发的、与市场上的大多产品都不径相同，他才能像一个有灵魂的生物一样，去主动探求和思考一些事情。

他数据库里显示着自己是量产化的，出厂时所有数据配置都一样。

但被下载到一个新电脑上时，他的内核却能自主书写新的异变代码。

——这是他与其他普通软件的最大区别。

他的资料里显示，他的存在，以及他的灵魂，应该算是『强人工智能』，他具有推理与解决问题的强大能力。

他甚至可以合理地使用与控制它们。

从他有意识以来，他称自己为“他”与“我”，而它们都是“它”而已。

这是他作为一个时代前沿技术产物的骄傲。

瞧。他还会用成语。

她无甚血色的嘴角叼着一朵新摘的雪白梨花，单手随随便便地扛着树枝方要气势十足地前行，却突然想起什么似的回头。

「……哦对惹，你们为什么总是变的一个比一个年幼，我记得你们早就能变成大人了叭。」

「……因为要向老大看齐。」

「我、我们怎么能比老大年长呢！」

「我们只是按老大的年龄比例计算而已。」

阿梨倏尔架起胳膊，低头盯起了抱着她小腿不放的三棵小树，眼神专注神色深沉，看的他们心里叮叮咚咚打起了小鼓。

她眨眨眼摸起了摸自己的下巴，然后一人给了一个轻轻的脑瓜锤。

「……给我变回去。」

「我们怎么能雇佣童工呢？」

「你们别欺负我睡的时间长年纪大了跟不上时代，这个基本常识我还是知道的口亨。」

「好的老大。」

「我、我马上就变。」

「……可是老大，你为什么不化形为成年女性的模样呢。」

三树纷纷抖落树叶转身踏入青绿色的灵力光晕里，只有小山楂迟疑了一步。

棕发的女高中生一愣，扛着的树枝不经意晃了晃，不小心脱力「咣当」掉到地上，砸出了一个三尺多深的土坑。

「……你管我！你们、快点换衣服，我先走了！」

她在对方有些探究的目光里手忙脚乱地将小木棍从泥里扯出来，佯装镇静地抬回肩膀跑了。

只留给他一个尘兮兮远去的背影与一地凌乱的碎雪。

【再不走出去，一别如斯，季节交迭，自己的花又要落干净了罢。】

【下一次，她上哪再去寻这样一个会对着枯干老木头说美的白痴呢。】

所以当药研少年伸伸僵硬不堪的腰肢，走出自己坚固防备却憋闷阴暗的地下室，不知时光流逝了多久，方打算出门透透早春清凉香气时，看到的场景就是——

一群人正向他家门口摆放着有一座小山那么高的水果堆。不仅果类花样繁多色泽鲜美，还各个丰硕圆润散发着沁人心脾的甘甜香气。

对面芬芳果山环绕中央，还站着一个棕发雪肤的女孩子，穿着和他极像的衬衫与背带短裤，正眉眼弯弯笑着向他招手，唇边扬起的弧度甜若梨花盛放，眉眼淡定如风拂掉枝桠上的积雪。

然而药研此刻打开一半的门显然比他的腰更加僵直。

他「咣」的一声立刻把门关上，心里默默想着大概是从地下室刚出来，头晕目眩未及清醒。

……又也许是他打开门的方式不对。

门内的轴体旋转出犹疑的弧度。

「喂！」

再次开门时已然近在眼前的微笑女生，眼角已经有要裂开的征兆。

「你居然敢把我关在外面，你——」

电光火石之间，药研一把将她拽进门内，绛紫的眼神犀利而严肃。

在药研没看到的角度，阿梨甫一挥手，将门外三棵树的所有行动阻了个严实。

「你不是木灵吗，怎么能随意跑到人间界来？」

他拉住她的手还没放开，覆着黑色手套的指尖微微用力，在雪色的肤上按出微浅的印，与她的距离不像往常一般亲切贴近，她闻不到他身上熟悉馥郁的木叶香气。

——他是又在责怪她不好好守护村民到处乱跑吗。

他向来疏朗坦荡却温和的眉目里，此刻透出了一丝刀锋的凛冽。

就如无机质的钢铁硬络地反射出尖峭的冷光。

……这分明是木头阿梨，最不喜欢的锋利模样。

掌心里幽幽冒起几朵火苗，绿眸泠泠转深，她皱起眉方要一把推开他。

「我想来便来！和你有什么关系——」

一只手，镇镇定定、正正好好地压上了她的发。

谁叹息一般的声音在头顶低朗响起。

「你啊，外面很危险的。」

谁生起的赤焰偃然间熄灭得悄无声息。

她抬头仔细地望向他，只见那双漂亮的藤色眼里曳满了温柔的无奈与担忧。

似乎方将一瞬的凌厉不过错觉，他又变回了那个认翻着草木香书册陪她度过一个又一个下午的爽朗少年。

「呐，阿梨——」

「叫我木灵大人。」

他尚未撤下手，而阿梨严辞端目纠正他的称呼后，却是毫无阻止他动作的意味，甚至还微微偏头垂下了眼睫，似是在享受人类手掌传来的温温热度。

「好好，我的木灵大人。」

少年倏然发觉她发尾不知何时悄悄绽开的绡白小花，嘴角淡淡扬起，但语气仍里是不容拒绝的硬朗坚定。

「我刚刚没有在开玩笑，城市里很危险，如果你没什么事情的话，还是在木林中活动比较安全。」

被顺毛正捋的舒服的阿梨，绿眸懒洋洋抬了抬，想了想自己还是解释清晰为好。

「这里对我来说也很安全。更何况……」

她轻轻举起面前黑发少年置于她发际的手，冰凉的指尖触碰到他温热的手心，仿佛冰凉的叶飘起了一湖涟漪。

药研眼睁睁地看着她神色认真将自己的手轻柔地扯到她怀里，拽出自己一根细如烟丝的发，细致地放于他手上。

「喏，给你。」

然后那根头发就变成了一个细长的木枝。

然后药研就感受到了几百斤的沉重。

若不是阿梨在一旁及时撑起了大半部分的重量，药研大概率就会被直接压跌在地上。

药研：「……」

「咳、你的实力，我知道了。」

侥幸逃过一劫的药研坦坦然收回了手，却似乎毫无「自己刚刚抚摸的也是什么重量级物体」的自觉，反而对着她……的头发露出了一个愉悦的笑容。

「不过……呐阿梨，你的头发，能给我几根吗。」

阿梨凝固：「……」糟了这个人可能坏掉了。

药研再次伸出手摸摸她，所之触及分明是柔软蓬松的绒发。

阿梨扭头不看他：「不给。」

语气里却全然又忘记了他称呼的细微变化。

紫眸的英气少年却忽然低头贴近，清冽的草药清香瞬间覆盖住了阿梨面庞，近在咫尺的鼻唇拂到她耳畔，与往常靠近她树本体的感觉却完全不同。

不再是粗糙的树表，人类细腻皮肤能直接感受到更加滚烫的热气。

比那时更加清晰可闻的、他沉稳又利落的呼吸声，不知为何让她想起尝过的最浓稠的蜜糖。

她这些年来，这是第一次有了，能化为人身真好的奇怪想法。

而安静半晌的药研终于开口，她只觉自己耳朵里甚至蔓延生长出了各色绮丽的花。

「呐真的，阿梨，你的发质柔软又亮丽，我很喜欢，能送给我几根吗？」

谁的木头心一下子跳动的异常热烈，她只觉得自己像是又着起了火。却不是自我伤害、燃烧树体生出的，是作为人类的内脏和血液，剧烈地鼓动与流窜，让她四肢百骸如同浸于地下翻腾的岩浆里。

烧灼至极，不能思绪。

「好。」

阿梨敛下眼睫，掩去慌乱，细蚊如促，如是说。

正当她方觉自己又甚没出息，应该推开他好好反驳一番的时候。

自己肩上却沉下一片暖热。

纤细修长的少年本就不重。

她轻轻松松揽住了他的后背。

却忽有不寻常湿热的触感，她手臂抬起。

——与记忆深处的鲜艳浓烈瞬间重合。

「喂——药研！」
