第一篇（序章）

\ \ \ \ \ \ \ \ \ \ \

bgm [The Rain - Vsun](http://music.163.com/song/493950499/?userid=264477003 ()

时间回溯。三天之前。

\ \ \ \ \ \ \ \ \ \ \

「下雨了。」

「嗯，要吃糖葫芦吗？」

\ \ \ \ \ \ \ \ \ \ \

窗沿里暗色的水珠窸静地汇聚，凌晨时分的深蓝城市扩散出霓色的潋滟靡丽。

「呐，咱们现在……是要做什么？」

窗影上出现一个举着双手的少年，由于身前的阻碍，头颈向后微微仰去，柔韧成一道白鹤般的优雅弧线。

只不过有时甚不老实地左摆右晃歪头就是了。

他面前正与围巾的一百零八种系法打交道以及与衣服的对称度持续战斗的女生头也不抬。

「准备杀时间溯行军啊。」

边说着她又将刚系好的围巾又拆了，端着下巴纳罕地打量起他。

「不对，这样不行。」

望着又要陷入纠结与深思的自家审神者，白发付丧神会意地放下胳膊，舒展地斜倒进一旁椅子里，趴在椅背上用透彻又清冽的金眸子瞧她。

「所以啊，时政发布的任务呢，为什么你这次什么都不说？」

「还是说……」

他淌了蜜的深秋色眼睛似笑非笑注视着在透蓝色屏幕上搜寻资料的她。

「你这次……是有什么难言之隐吗？」

女生飞快移动着的眼球有一瞬凝滞。

「果然……鹤先生还是长的太好看了，」她盯视住一张图片使其自动增长分辨率后，语调突然升高，透出平素语气里未曾掺过的嫌弃，「我明明已经按这个时代的最低要求来搭配了，结果……」

她的脸忽然在面前放大，手指拎起他的额边碎发后却沿着发线逐渐下移，刀削的冰刻下巴被她温热的手指轻轻点触。

她专注认真又毫无机质的深黑瞳孔里，曵起了屋外淅沥的雨声。

「呐，近侍大人……我可以擅自改变你的模样吗。」

容貌昳丽的白色少年沉静地盯了她几许，随即就微眯眼睛疏狂地笑起来。

「嘛……既然你这么有自信的话，我就尽情享受惊喜了。」

她淡淡的目光里亦映出了温意。

「承蒙信任，不胜感激。」

「那么还是请您先站起来，让我把这个围巾处理妥当吧。」

「喂、等等……这个，你已经试了109种方法2个小时了还要再来吗……」

「你刚刚才说过信任我的。」两分委屈。

「……好好好，我任你宰割还不行吗。」铺伸开来的白羽翅，在昼色灯下亮的洁白晃眼。

「用词不当，请将宰割去掉。」羊绒厚实的围巾，威胁似得软绵绵裹紧了白皙的脖颈。

「……我任你摆布。」金眼睛弯成溪流旁亮晶晶的细沙。

「好，我同意了。」她有些孩子气地叉起腰。

溪流里的金沙，亦仿佛流的更欢了。

\ \ \ \ \ \ \ \ \ \ \

一抹珍珠白旋转着出现在色彩纷繁的众伞中央。

举着伞柄的手是健康的小麦色，随着行走与风拂而微微扬起的是鸦黑的短发。时刻蕴着温笑的眼睛里反射出的是墨色琉璃的光彩。

「这可真是吓到我了，你确定你真的不会遭到灵力反噬吗？」

少年如水滴一样透亮的声音清凉凉钻进身旁同样乌眉黛眼人的耳里。

她眉眼自信地扬起。

「没问题的，这是假象。我只是利用灵力暂且改变了你身上一些东西的频率，不过是糊弄人眼的东西而已。」

「不过……鹤先生真是难办啊，就算变成这样低调的样子，却还是能吸引住很多纯情少女的目光呢。 」

她带着笑意和揶揄看向他身后不少闪着惊艳的亮晶晶眼神。

「喂喂，饶了我吧，你把我的头发布置的连脸都快遮没了，而且我穿的不也是你所谓的大叔装吗～」

他向外竖了竖土色工装外套的立领，却不小心扯松了扣子，深色的内衬与美学顶级形状的锁骨被扯现出来。虽已变成低调不再雪白流光的小麦黄肤色，却反而让他生出一种平素不曾显露的野性吸引力。

她默了默，无奈地还是伸出手决定替他系回去。

他微屈身配合她，一侧没握着伞的手将她故意拨到他额前的几丝碎发轻撩开来，向她轻快地眨眨眼：「有什么关系，反正时政会把他们的记忆都替换掉不是吗？」

不同以往的墨色眼睛于近在咫尺的地方清澈透亮，浅淡鎏金色火焰自那丝绒一般的深色幕布下微微风动。

奇怪。

怎么她的引擎就没搜索到，这人瞳色变深之后，原本淡色瞳孔能掩盖的情绪，现在似是遮不住了。

轻快、俏皮、戏谑、却又专注、镇定、温柔，仿佛全世界就只能看见她一个人的、一种奇特异样的隐藏情绪。

正被他眼眸里浓的落墨的色调。

——点点滴滴地出卖着。

时空有瞬间的寂静，谁耳边叮咚的雨声仿佛被莫名生长的屏障隔离殆尽。

一句不知怎得生出、想要强烈表达出的『有关系』在喉咙里来回窜行了数次。

最终还是消弭于胸腔深处。

她只是一本正经地开口。

「……总是人为干预大脑的记忆，他们会变傻的。」

「哦？原来是这样，那么我就照顾一下他们好了。」

眼见面前少年模样的神祗开始十分自觉地回揽自己的衣襟与领扣，她低下头藏起自己低低翘起的嘴角。

她点触手上腕表查看时间，恰好又错过头上温热悠长投来的沉金色。

屏幕灵敏飞快地切换，最终定格于指向圆周四分之三处的莹蓝指针。

再抬起头时她已是略有歉色的笑意，他也迅速转回四处张望的视线温温看她。

「不好意思鹤先生，上课时间要到了。」

说完她握住旁边那人故意歪向她大半的伞柄下侧，边向反方向微微推动，边向前轻拽起尾端。

她加快步伐回头笑起来。

「鹤大叔，再不快点，我就要把伞抢走了哦。」

\ \ \ \ \ \ \ \ \ \ \

宽敞明亮的阶梯教室里虽已喧闹一片，陆续走进的人身上却也多少沾着秋霖的冰凉，浓度适宜的烟火气一寸寸温然蔓延开来。

鹤丸国永亦将手里皎白的伞学着众人的样子撑开落于教室外的空阔处。

调整好她安置在自己耳里的隐形助译器，确认脑海里进入产出的都是上世纪的标准语。思维中回荡着那人方才递过来的寥寥信息，他扯出一抹无奈的笑意。

『嘛，无论如何，让鹤来和20岁的异国年轻人称兄道弟……也太过于相信我这个平安老头子了吧。』

然而他转过身来就径直向前面一排的五个人爽快地凑了过去，拍拍其中一个人的肩膀。

「哟！不好意思我来晚了，怎么样，老师还没来吧？」

那人笑呵呵搂住他肩膀，他还更加亲热地扶了回去。

「诶呀是鹤丸啊～」

另一边的少年温和地向他笑笑。

「坐吧鹤丸，老师还没来。」

还有一个正奋笔疾书的，见他来眼睛一亮。

「诶鹤丸老兄，上周的作业做的怎么样？」

他微愣几秒，随即淡定回复。

「哦那个啊！……做完了，还算简单，怎么了？」

「是嘛是嘛！来和我对一下吧——」

「没问题～」

对21世纪年轻大学生领域一无所知的鹤丸国永同学，捧着自家审神者事先准备好的一大摞资料。对着五个名义上的舍友，笑容可掬地胸有成竹。

\ \ \ \ \ \ \ \ \ \ \

距离上课还有半分钟。

和『舍友们』相谈甚欢的鹤丸忽然似有所感地转头。

精准地踏着清脆急促的铃音，一个身影凌冽地翻了进来。

坐在靠边第二个位置的鹤丸下意识敞开了臂膀。

满怀的霜冷风雨与温热柔软。

他一旁的少年眼尖地发现了他怀里多出来的人，笑眯眯地将大半部身子挪到左边自动消音也将好奇的其他人隐了视线。

「好了好了老师还没来，咱一起上会自习别看了别看啦～」

鹤丸感激地看了对方一眼。

回眸发现怀中人浑身渗发着的冰雪气似乎越来越浓，甚至有蔓延加剧的态势。而体重似乎也愈来愈轻，几近于无。

他方要低声询问她如何了，却见她紧闭的黑眸遽然而睁。

异样皆数消失。

她尚还脱力趴于他胸前，却忽清清淡淡笑起来。

「怎么了？抱我抱上瘾了吗。」

鹤丸国永默而不语盯了她许久，直至门外高跟鞋的啪嗒声叠叠响起。

他才敛起再次不小心流露碎金的黑瞳，轻轻将她扶到桌上稳定借力。

「下次……不必这么拼。」

她将气力恢复些的身姿微微调整坐正时，耳边低低柔柔钻进了有些气馁的音节，额边发丝被微凉的指尖徐徐地挽起。

她看向近在咫尺、表情里难得透出肃穆又挫败的颜色的他。

眼睛扑闪两下，眨出亮痕，反而扯起唇角笑了。

「好。」
